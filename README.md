Personal Portfolio Site

Live Site Link: https://capstone-frontend-86m7.onrender.com

In this project, I built a responsive personal portfolio website using HTML, CSS, JavaScript, React, Node.js, Express, and MongoDB. I used Postman for API testing, Cloudinary for media management, and deployed the site via Render. 



Technologies Used: HTML, CSS, JavaScript, React, Express, Node.js, MongoDB, AJAX, Postman, Render, Cloudinary
